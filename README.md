# 使い方
わかりやすくレイヤーに色付などをしています。
カメラ制御のエフェクトが完備されていて、グループ制御のエフェクトのチェックのONOFFで設定できます。
# ダウンロード
画面右の「Release」を押して、下側の青いymmpファイルをダウンロードしてYMMにドラッグ&ドロップしよう！
